Support Center

We're Here to Help

Welcome to the Basta Community Support Center. Our goal is to provide reliable assistance and ensure you have the best experience with our products. Whether you need help using an application, have encountered a technical issue, or would like to suggest a new feature, we're always happy to hear from you.

Products We Support

Paperleaf

Support for drawing tools, sketchbooks, notebooks, project management, backups, premium features, and future cloud services.

PageFlip Library

Technical support and documentation for developers using our page-turn animation library.

Paperbook

Support for creating, publishing, purchasing, and reading digital flip books. (Available after release.)

Frequently Requested Help

We can assist you with:

  • Installation and setup
  • Account and sign-in issues
  • Restoring purchases
  • Premium feature activation
  • Importing and exporting projects
  • Performance optimization
  • Cloud synchronization
  • Reporting bugs
  • Requesting new features
  • General application questions

Report a Problem

If you encounter a bug or unexpected behavior, please include the following information when contacting us:

  • Application name
  • App version
  • Android version
  • Device model
  • Description of the issue
  • Steps to reproduce the problem
  • Screenshots (if available)

Providing detailed information helps us investigate and resolve issues more quickly.

Feature Requests

Many improvements begin with suggestions from our users. If you have an idea that could improve Paperleaf, Paperbook, or any other Basta Community product, we'd love to hear it.

While we can't guarantee every request will be implemented, all feedback is carefully reviewed and considered for future updates.

Development Status

Our products are actively evolving. New features, improvements, and optimizations are released regularly as development progresses.

For the latest announcements and release notes, please visit the News page.
